## Technical Decisions
| Decision | Rationale | Alternatives |
|----------|-----------|-------------|
| HMAC-signed cookies | No new dependency needed, crypto built-in | iron-session, JWT |
| Atomic XP increment | Prevents race condition in concurrent events | Pessimistic locking |
| Stories batch-fetch views | O(1) query vs O(n) per story | Include all views (memory issue at scale) |
| Seed in instrumentation.ts | Runs once on startup, idempotent upserts | Separate seed command |
